# Sales Dashboard in Excel

## Project Overview

This project demonstrates the creation of an interactive Sales Dashboard using Microsoft Excel.

## Dashboard Features

* Total Sales KPI
* Total Profit KPI
* Total Orders KPI
* Profit Margin %
* Monthly Sales Trend
* Sales by Region
* Sales by Category
* Top Products Analysis
* Interactive Filters and Slicers

## Tools Used

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Slicers
* Conditional Formatting
